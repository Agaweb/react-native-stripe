# react-native-stripe

Stripe payments for react-native

## Installation

```sh
npm install @agaweb/react-native-stripe
```

## Usage

```js
import Stripe from "@agaweb/react-native-stripe";

// ...

const result = await Stripe.multiply(3, 7);
```
